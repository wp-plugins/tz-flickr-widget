Plugin has get your Flickr photostream in a sidebar easily without authentication.

== Description ==

This is a simple widget to display your Flickr latest photostream in widget area using Pretty Photo to open gallery images.
You can get your Flickr ID at http://idgettr.com/.
With its very beautiful widget interface, tabbed system, and powered by jQuery makes this plugin easier to customize.
Just put your Flickr ID and your widget will be ready to lunch.

<h3><strong>Features & Options</strong></h3>
<ul>
<li>Beautiful widget interface</li>
<li>The type of images from user id.</li>
<li>Get the latest image.</li>
<li>Put your number images you want to display.</li>
<li>Optional image sizes.</li>
<li>PrettyPhoto for image thumb.</li>
<li>Free supports.</li>
</ul>

== Installation ==

1. Go to the menu 'Plugins' -> 'Install' and search for 'TZ Flickr Widget'
2. Click 'install'

== Frequently Asked Questions ==
= How to use Pretty Photo to view gallery? =
You go to TZ Flickr Widget Dashboard and checked in 'Use Pretty Photo'
